# Kybernesis Claude Code Plugin

Connect your Kybernesis AI agents to Claude Code. Chat with personalized AI agents that have access to your workspace memories.

## Installation

1. Add the Kybernesis marketplace:
   ```
   /plugin marketplace add ianborders/kybernesis-claude-code-plugin
   ```

2. Install the plugin:
   ```
   /plugin install kybernesis@kybernesis
   ```

## Setup

Set your Kybernesis API key as an environment variable:

```bash
export KYBERNESIS_API_KEY="kb_your_api_key_here"
```

Get your API key from: https://kybernesis.ai/settings/api-keys

## Usage

Once installed, Claude Code will automatically know how to interact with your Kybernesis agents. You can:

- Ask Claude to list your agents
- Talk to specific agents by name
- Have agents search your workspace memories

## Features

- **Agent Chat**: Chat with any of your Kybernesis agents
- **Memory Access**: Agents can search and retrieve relevant memories from your workspace
- **Conversation Continuity**: Maintain context across multiple messages
- **Multi-Agent Support**: Switch between different agents as needed
